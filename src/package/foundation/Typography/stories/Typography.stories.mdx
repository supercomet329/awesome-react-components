# Typography

This component can be used to render all text-based elements on
a page such as headings, paragraphs, captions, and labels.

## Usage

```tsx
import { Typography } from "hazel-ui";

function Component() {
  return (
    <Typography variant="display">
      Could a computer ever create better art than a human?
    </Typography>
  );
}
```
